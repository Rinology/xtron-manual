# 퀄리 전기자전거 웹 매뉴얼 관리 가이드 (초보사용자용)

이 프로젝트는 누구나 쉽게 새로운 매뉴얼을 추가하고 관리할 수 있도록 설계되었습니다. 
코딩을 잘 모르시더라도 아래 단계만 차근차근 따라 하시면 새로운 내용을 추가하실 수 있습니다.

---

## 🚀 실시간으로 수정 내용 확인하기 (라이브 뷰)

수정한 내용을 내 컴퓨터에서 실시간으로 확인하려면 아래 명령어를 사용하세요.

### 📝 실행 방법:
1. VS Code 하단 터미널(Terminal)창을 엽니다. (단축키: `Ctrl + `` )
2. 아래 명령어를 입력하고 엔터를 누릅니다.
   ```bash
   npm run dev
   ```
3. 터미널에 뜨는 주소(예: `http://localhost:5174/`)를 `Ctrl + 클릭` 하거나 브라우저에 입력하면 실시간 화면이 열립니다.
   * **팁**: 코드를 수정하고 저장(`Ctrl + S`)하면 브라우저 화면이 자동으로 새로고침되어 반영됩니다!

---

## 1. 간단한 매뉴얼 추가하기 (방법 A)
별도로 다른 파일을 만들 필요 없이, 기존 목록에 한 줄을 추가하는 가장 쉬운 방법입니다.

### 📝 따라하기 단계:
1. **`src/data/guides.jsx`** 파일을 엽니다.
2. `items: [` 라고 적힌 부분(약 11번째 줄 등)을 찾습니다.
3. 아래의 **[복사용 코드]** 전체를 복사합니다.
4. 기존에 있던 항목의 끝부분( `},` ) 바로 아래에 붙여넣습니다.
5. 붙여넣은 내용 중 `"쌍따옴표"` 로 감싸진 글자들만 원하는 내용으로 수정합니다.

### [복사용 코드]
```javascript
{
  id: "my-new-manual-01",      // (중요) 영어로 중복되지 않게 이름을 지어주세요.
  title: "여기에 제목 입력",      // 화면에 보일 실제 제목입니다.
  icon: <Wrench size={18} />,  // 아이콘입니다. (그대로 두셔도 됩니다)
  summary: [                   // 검색 결과에 보일 요약 문장들입니다.
    "첫 번째 단계: OO를 확인하세요.",
    "두 번째 단계: XX를 조입니다.",
    "세 번째 단계: 완성을 확인하세요."
  ],
  placeholderType: "image",    // 사진이면 "image", 움직이는 사진이면 "gif"
  placeholderText: "사진이 없을 때 나올 설명글" 
},
```

## 2. 나만의 상세 페이지 만들기 (방법 B - 고수용)
'페달 장착 방법'이나 '타이어 공기압 체크'처럼 사진과 긴 설명, 주의사항 등이 들어가는 예쁜 전용 페이지를 새롭게 추가하는 방법입니다. 단 2단계만 거치면 됩니다.

### 🚀 1단계: 내용물이 담길 컴포넌트 만들기 (`src/components/` 폴더)
1. **새 파일 만들기**: `src/components/` 폴더 안에 새로운 파일을 만듭니다. (파일 이름 끝에 `.jsx`를 꼭 붙이세요! 예: `MyManual.jsx`)
2. **코드 붙여넣기**: 기존에 있는 `src/components/PedalGuide.jsx` 파일의 코드를 복사해서 새 파일에 그대로 붙여넣습니다.
3. **내용 수정하기**: 복사된 코드 안에서 함수 이름(예: `export default function MyManual`)과 본문의 제목, 설명 텍스트, 아이콘만 새 주제에 맞게 수정하여 틀을 완성합니다.

### 🔗 2단계: 웹사이트 메뉴판에 등록하기 (`src/data/guides.jsx` 파일)
1. **파일 위쪽에서 불러오기(Import)**: `src/data/guides.jsx` 파일의 맨 위쪽(import가 모여있는 곳)에 1단계에서 만든 파일을 불러옵니다.
   ```javascript
   import MyManual from '../components/MyManual';
   ```
2. **원하는 카테고리에 내용 추가하기**: 파일 중간에 있는 `categories` 배열을 살펴보고, 넣고 싶은 곳(예: `id: "basic"`)의 `items: [` 안에 아래처럼 새 내용을 적어줍니다. 기존 양식을 복사해서 수정하면 쉽습니다.
   ```javascript
   {
     id: "my-new-manual",            // 영어로 된 고유 이름 (URL이나 고유값으로 쓰임)
     title: "사이드바에 뜰 새 제목",      // 사이드바 메뉴에 표시될 제목
     icon: <Wrench size={18} />,     // 메뉴 옆에 표시될 아이콘
     summary: [                      // 검색 시 보여줄 요약 설명 (따옴표로 묶어 작성)
       "첫 번째 요약 설명",
       "두 번째 요약 설명"
     ],
     placeholderType: "image",
     placeholderText: "안내 이미지 공간",
     customComponent: MyManual       // ★중요★ 1단계에서 만든 컴포넌트 이름 연결!
   }
   ```
이렇게 추가하고 파일을 저장(`Ctrl+S`)하여 다시 GitHub에 `push`하면, 실제 웹사이트 메뉴에도 생기고 클릭 시 방금 만든 상세 페이지가 보입니다!

## 3. 사진과 GIF(움직이는 사진) 넣는 법

### 📸 1단계: 사진 파일 넣기
1. 사진 이름을 영어로 짓습니다. (예: `test.jpg`)
2. **`public/images/`** 폴더 안에 사진을 넣습니다. 
   - (이미 `pedal`, `battery` 폴더를 만들어 두었으니 관련 사진은 해당 폴더에 넣으시면 관리가 편합니다!)

### 🔗 2단계: 코드에서 보여주기
글을 쓸 때 사진을 보여주고 싶다면 아래 한 줄만 적으시면 됩니다.
```html
<img src="/images/내파일명.jpg" />
```

## 💡 이것만은 꼭! (주의사항)
*   **콤마(`,`) 조심!**: 항목과 항목 사이에는 반드시 쉼표가 있어야 합니다.
*   **소문자 사용**: 파일 확장자는 반드시 소문자 `.jpg` 여야 합니다. (대문자 `.JPG`는 인식을 못 할 수 있어요!)
*   **괄호 짝맞추기**: `{`를 열었으면 반드시 `}`로 닫혀 있는지 확인해 주세요.
